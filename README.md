# bouncer

> A GitHub App built with [Probot](https://github.com/probot/probot) that Auto approves of github action jobs that use environments to access secrets based on branch and submitter.

## Setup

```sh
# Install dependencies
npm install

# Compile typescript
npm run build 

# Run the bot
npm start
```
